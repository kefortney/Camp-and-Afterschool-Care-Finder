Camp & Afterschool Care Finder

An open-source tool helping Vermont families find summer camps and afterschool programs for K–12 students.

About the Project

This was originally created by parents of a pre-schooler in the Burlington area who suddenly needed to find Summer Camps for that age range and were started to find no easy way to do so. Two years of doing this research manually in Google Sheets (and sharing it with like parents) led to the idea of building a tool to make this process easier for other families in the future.

The Camp & Afterschool Care Finder is a free, publicly accessible, community supporter web tool built for Vermont families. It aggregates information about summer camps and afterschool programs serving kindergarten through 12th-grade students across Vermont. Families can filter by grade level, city, program type, cost, subject area, and schedule to quickly find programs that fit their child and their budget. Programs can be browsed as a list, on a calendar, or on an interactive map.

The project is entirely open source. All data is collected from public sources and is freely available in this repository (see the GitHub repository ).

Why This Matters

Finding a summer camp or afterschool program in Vermont is harder than it should be. There is no central registry or statewide database for summer programs. Each organization maintains its own website, or doesn't have one at all, and information about availability, cost, grades served, and registration deadlines is scattered across dozens of disconnected sources. And for any parent who has the pleasure, these camps are filled quickly, so early registration within hours of them being posted is crucial

The result: families who have the time and resources to do extensive research can find great programs, while families with less time or less familiarity with navigating websites often miss out entirely. Lower-income or time-constrained families are disproportionately affected, as subsidized and free programs like 21st Century Community Learning Centers are often unknown even to the families who need them most.

Geographic access is also unequal. Programs in Burlington are well-documented compared to programs in rural Vermont towns, and the lack of visibility for smaller programs makes it harder for them to reach capacity even when spots are available.

This project is a first step toward a more equitable information landscape for Vermont families. The outputs: a web app and a dataset are just first steps toward that goal. The initial hope of solving a collective action problem in creating a more accessible and transparent information ecosystem will hopefully lead to questions on how we can grow and expand Camps and Afterschool Care for working families.

Contribute on GitHub

This project is open source and welcomes contributions of all kinds — from adding missing programs to the dataset, to improving the search UI, to fixing bugs and improving documentation.

How to Contribute

  1. Fork the repository on GitHub and clone it locally.
  2. Add or improve data — the easiest contribution is researching missing programs and adding rows to data/programs.csv or verifying existing records against current program websites.
  3. Run the data pipeline — after editing CSVs, run python scripts/validate_data.py then python scripts/build_data_js.py to regenerate data.js.
  4. Test locally — run python -m http.server 8000 and open http://localhost:8000.
  5. Submit a pull request with a description of what you added or changed.

New to open source? Check the Issues board for tickets labeled good first issue. All experience levels are welcome.

The Data

All data lives in two CSV files. A Python pipeline validates and joins them into data.js, the module loaded by the app. See the Data page for full field-by-field documentation.

Programs — data/programs.csv

The unified program registry: 401 programs covering 391 summer camps and 10 afterschool programs. Each row is one program session with dates, grades served, hours, cost, location, activities, and a description. After editing, run python scripts/validate_data.py then python scripts/build_data_js.py to regenerate data.js.

Organizations — data/organizations.csv

The organization registry: 148 organizations that run programs in the dataset. Provides website, phone, email, address, financial aid availability, and registration policy. Every program row references an org_id that must match a record here — enforced by validate_data.py before building.

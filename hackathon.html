The Challenge

Every summer, thousands of Vermont families scramble to find camps and programs for their kids — and the experience is far harder than it needs to be. There is no central registry for Vermont summer programs. No searchable statewide database. No single place to compare costs, grades served, or available schedules.

Instead, families must piece together information from dozens of individual program websites, Facebook pages, school newsletters, and word of mouth. Families with more time and more digital fluency consistently find better programs. Families with less — often lower-income families who stand to benefit most from subsidized programs — too often miss out entirely.

This project is an attempt to fix that. We're building an open, free, searchable tool for Vermont families — and we need your help to make it better.

Why This Data Is Hard to Gather

If you're wondering why we don't just "scrape it all" — here's what makes this problem genuinely hard:

No central registry for summer camps

Vermont licenses childcare providers, which creates a public dataset. But summer camps are not licensed the same way, so no equivalent database exists. Every program is its own island.

Wildly inconsistent data formats

Some programs post structured registration forms. Others publish PDFs. Some post on Facebook only. A few send home paper flyers. Cost might be listed as "$250/week", "$50/day", "sliding scale", or "call us." Dates might be "June 23 – July 4" or "summer 2026" or nothing at all.

Data goes stale quickly

Programs change their dates, drop sessions, add new programs, or close entirely from year to year. Maintaining accuracy requires re-verifying every record annually — a time-consuming manual process.

Many programs have minimal web presence

Smaller programs — especially those serving rural communities — may have no website, or a website that hasn't been updated in years. Finding accurate information requires direct phone or email outreach.

Geocoding and address normalization is manual-intensive

Address formats are inconsistent, many programs share a school or community center address rather than their own, and some programs have no fixed location at all. Placing them accurately on a map requires human judgment.

Missing geographic coverage

Our current dataset covers the Burlington area reasonably well, but large parts of Vermont — particularly rural northeastern and southeastern counties — are significantly underrepresented. Filling those gaps takes local knowledge.

Ways to Contribute

Whether you're a developer, a researcher, a designer, or just someone who knows Vermont programs — there's a way for you to help.

Add missing camps

Research programs not yet in the dataset and add rows to data/programs.csv. Rural Vermont towns are especially underrepresented.

Add towns to the map

The map view uses a hardcoded coordinate lookup (CITY_COORDS in app.js). Many Vermont towns are missing — add their lat/lng to show programs on the map.

Improve data quality scripts

The scripts/ directory has Python tools for address enrichment, cost parsing, and description fetching. There's a lot of room to make them smarter.

Improve the search & filter UI

Better mobile layout, smarter autocomplete, a "near me" feature, or improvements to the map popups — UX improvements go a long way.

Add more afterschool programs

Only 10 afterschool programs are currently in the dataset. Research other Vermont afterschool providers and add them to data/programs.csv with program_type = afterschool.

Improve cost normalization

Cost data in data/programs.csv is stored both as raw text (cost_raw) and a normalized weekly number (cost_per_week). Many rows have missing or inconsistent cost_per_week values — filling these in makes the cost filter more reliable.

Fix bugs & open issues

Check the GitHub Issues board for known bugs and feature requests. Many are small, well-scoped, and good for a hackathon sprint.

Documentation

Improve the README, add inline code comments, or write a contributor guide to help future contributors onboard faster.

Get Started

Everything you need to contribute is in the GitHub repo. No login, no special access — just fork and go.

Quick Setup

  1. Clone or fork the repo: git clone https://github.com/kefortney/Camp-and-Afterschool-Care-Finder.git
  2. Start a local server: python -m http.server 8000 (or python scripts/dev_server.py for live reload)
  3. Open http://localhost:8000 in your browser
  4. Edit files, reload, repeat — no build step required

Working with Data

  1. Edit data/programs.csv or data/organizations.csv in Excel, Google Sheets, or any text editor
  2. Run python scripts/validate_data.py — fix any errors it reports
  3. Run python scripts/build_data_js.py to regenerate data.js
  4. Reload the browser to see your changes
